[[span-first-query-usage]]
== Span First Query Usage

=== Fluent DSL Example

[source,csharp]
----
q
.SpanFirst(c => c
    .Name("named_query")
    .Boost(1.1)
    .Match(sq=>sq
        .SpanTerm(st=>st.Field(p=>p.Name).Value("value"))
    )
    .End(3)
)
----

=== Object Initializer Syntax Example

[source,csharp]
----
new SpanFirstQuery
{
    Name = "named_query",
    Boost = 1.1,
    End = 3,
    Match = new SpanQuery
    {
        SpanTerm = new SpanTermQuery { Field = "name", Value = "value" }
    }
}
----
